# 领域验收：订单履约

**适用**：订单创建、状态流转、取消、超时、部分发货、售后等场景。

---

## 1. 订单创建与状态机

| 验收点 | Then 判据示例 |
| --- | --- |
| 订单创建 | 订单创建成功；订单号唯一；状态为「待支付」 |
| 状态流转 | 订单状态按 spec 定义的合法路径流转；无非法状态跃迁 |
| 超时取消 | 超时未支付订单自动关闭；库存回滚 |

---

## 2. 部分发货与库存

| 验收点 | Then 判据示例 |
| --- | --- |
| 部分发货 | 订单支持部分发货；已发/未发商品状态分离 |
| 库存扣减 | 发货时库存正确扣减；超卖防护生效 |
| 售后入口 | 已发货订单可申请售后；售后流程与 spec 一致 |

---

## 3. 取消与退款

| 验收点 | Then 判据示例 |
| --- | --- |
| 用户取消 | 未发货订单可取消；库存回滚；状态正确 |
| 取消权限 | 取消操作权限与 spec 一致；无权限不可取消 |
| 取消后退款 | 取消后退款原路返回；金额正确 |

---

## 4. 履约时限与通知

| 验收点 | Then 判据示例 |
| --- | --- |
| 履约时限 | 订单履约在约定时限内完成；超时预警生效 |
| 履约通知 | 订单状态变更通知用户；通知渠道与 spec 一致 |
