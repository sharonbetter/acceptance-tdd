# 通用验收：页面操作与手势

**适用**：桌面点击、键盘、移动端触摸、拖拽排序、长按菜单、可访问性操作。

---

## 1. 点击与双击

| 验收点 | Then 判据示例 |
| --- | --- |
| 主按钮 / 次要按钮 | 触发正确动作；loading 中防重复提交 |
| 双击防抖 | **无**双下单、双次导航（以 spec 为准） |
| 热区与禁用态 | 禁用不可点；可访问性焦点顺序合理 |

---

## 2. 滑动与滚动

| 验收点 | Then 判据示例 |
| --- | --- |
| 列表惯性滚动 | 触底加载与 `acceptance-general-batch-pagination.md` 一致 |
| 横向滑动（Tab/Carousel） | 边界回弹、当前页指示正确 |
| 滚动位置恢复 | 返回上一页是否恢复滚动条（按 spec） |

---

## 3. 拖拽

| 验收点 | Then 判据示例 |
| --- | --- |
| 排序拖拽 | 放手后顺序持久化；取消拖拽恢复原序 |
| 拖入无效区域 | 高亮拒绝或回滚；**无**静默丢失 |
| 拖拽与滚动容器 | 不误触页面滚动 |

---

## 4. 长按与上下文菜单

| 验收点 | Then 判据示例 |
| --- | --- |
| 长按触发延迟与反馈 | 与 spec 一致；**无**与系统长按冲突未处理 |
| 菜单项权限 | 无权限项不展示或禁用（与 `acceptance-general-permissions.md` 一致） |

---

## 5. 键盘与辅助操作

| 验收点 | Then 判据示例 |
| --- | --- |
| Enter 提交、Esc 关闭 | 与表单/弹窗 spec 一致 |
| 焦点陷阱（Modal） | `Tab` 不逃出 Modal 直至关闭 |

---

## 6. 响应信息与反馈

| 验收点 | Then 判据示例 |
| --- | --- |
| 成功 / 失败 Toast、内联错误 | 文案与码一致；**可观察** |
| 触觉 / 声音（若有） | 不重复、不遗漏；可关闭时遵守系统设置 |

---

## 7. 自动化注意

- **E2E**：优先稳定选择器；手势用 Playwright/Appium 支持的真实用户路径。
- **Then**：断言 **UI 状态或网络结果**，而非仅「事件触发」实现细节
