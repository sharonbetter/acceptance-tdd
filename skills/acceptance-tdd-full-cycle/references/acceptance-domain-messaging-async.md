# 领域验收：消息与异步

**适用**：消息队列、异步任务、事件通知、重试补偿等场景。

---

## 1. 消息生产与消费

| 验收点 | Then 判据示例 |
| --- | --- |
| 消息生产 | 消息格式符合契约；关键字段完整；不泄露敏感信息 |
| 消息消费 | 消费成功/失败处理正确；消息不丢失、不重复消费 |
| 消息顺序 | 有序消息消费顺序与生产顺序一致；乱序处理符合业务规则 |

---

## 2. 重试与补偿

| 验收点 | Then 判据示例 |
| --- | --- |
| 重试策略 | 重试次数、间隔符合配置；重试失败死信处理正确 |
| 补偿与回滚 | 异步任务失败时，前置操作补偿/回滚正确 |
| 幂等处理 | 重复消息不重复处理；幂等键生效 |

---

## 3. 消息延迟与定时

| 验收点 | Then 判据示例 |
| --- | --- |
| 延迟消息 | 延迟消息按时投递；不提前、不延迟 |
| 定时任务 | 定时任务按时触发；任务执行结果符合预期 |
| 消息堆积 | 消息堆积时系统稳定；堆积处理符合业务规则 |

---

## 4. 权限与观测

| 验收点 | Then 判据示例 |
| --- | --- |
| 消息权限 | 不同角色/用户可消费的消息不同；权限控制生效 |
| 日志与监控 | 消息生产、消费、重试等日志可追溯；监控指标正确 |
