# 领域验收：广告与推荐

**适用**：涉及「广告 / 推荐」时展开。**计费曝光**若与统计口径相关，须在 spec **一处写清定义**并与数据域对齐，他处仅引用。

---

## 1. 广告展示与合规

| 验收点 | Then 判据示例 |
| --- | --- |
| 素材合规 | 图片/文案符合平台规范；无违规内容、敏感词 |
| 频控与排期 | 同一用户展示次数符合配置；时段排期生效 |
| 屏蔽与偏好 | 用户屏蔽词/品类生效；不展示已屏蔽内容 |

---

## 2. 推荐算法与排序

| 验收点 | Then 判据示例 |
| --- | --- |
| 冷启动与热数据 | 新用户/冷启动推荐策略生效；热数据不影响冷启动 |
| 个性化与多样性 | 不同用户推荐列表差异明显；不重复推荐同类内容 |
| 兜底与降级 | 无数据/算法异常时展示兜底内容；不展示空白 |

---

## 3. 曝光与点击计费

| 验收点 | Then 判据示例 |
| --- | --- |
| 曝光/点击计数 | 符合平台统计口径；不重复、不遗漏 |
| 防作弊 | 无效点击（机器人、刷量）不计费；作弊行为可定位 |
| 数据对账 | 广告平台与业务平台数据一致；对账无差异 |

---

## 4. 权限与风控

| 验收点 | Then 判据示例 |
| --- | --- |
| 广告位权限 | 不同角色/用户看到的广告位不同；权限控制生效 |
| 风控与反作弊 | 异常流量/IP 被拦截；风控日志可追溯 |
