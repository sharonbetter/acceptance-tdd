# 领域验收：订阅与计费

**适用**：订阅创建、周期续费、试用转正、退款、发票等场景。

---

## 1. 订阅创建与周期

| 验收点 | Then 判据示例 |
| --- | --- |
| 订阅创建 | 订阅创建成功；周期、金额、生效时间与配置一致 |
| 试用转正 | 试用到期后按时转正或停止服务；通知及时 |
| 周期边界 | 周期边界计费正确；不重复扣费 |

---

## 2. 续费与支付

| 验收点 | Then 判据示例 |
| --- | --- |
| 自动续费 | 续费扣款成功；订阅状态正确延续 |
| 续费失败 | 续费失败后重试策略与 spec 一致；通知用户 |
| 支付方式变更 | 支付方式变更后下次扣款使用新方式 |

---

## 3. 退款与冲正

| 验收点 | Then 判据示例 |
| --- | --- |
| 退款政策 | 退款金额与政策一致；不退现金时账户余额正确 |
| 周期退款 | 提前取消退款计算正确；按剩余周期比例 |
| 冲正 | 重复扣款后冲正正确；账户余额回滚 |

---

## 4. 发票

| 验收点 | Then 判据示例 |
| --- | --- |
| 发票申请 | 发票申请内容与实际消费一致 |
| 发票开具 | 发票开具后状态正确；不可重复开具 |
| 发票内容 | 发票明细（金额、税率）与订单一致 |
