# 通用验收：页面功能与数据流

**适用**：Web SPA、多页应用、混合 App 内 H5。

---

## 1. 链接与路由

| 验收点 | Then 判据示例 |
| --- | --- |
| 内部路由 / 外链 | `href` 正确；外链 `rel`/新窗口行为符合 spec |
| 深链与浏览器刷新 | 刷新后仍落在正确页面与数据；**无**白屏需手动回首页 |
| 404/无权限路由 | 统一占位页或重定向策略与 spec 一致 |

---

## 2. 参数传递

| 验收点 | Then 判据示例 |
| --- | --- |
| Query / Path 参数 | 缺参、非法类型、越界值：校验与错误展示符合契约 |
| 敏感参数 | **不**出现在可分享 URL 中（若 spec 禁止）；或脱敏展示 |

---

## 3. 组件状态

| 验收点 | Then 判据示例 |
| --- | --- |
| 加载/空/错三态 | 各态 UI 与禁用逻辑正确；错误可重试 |
| 表单脏状态与离开提示 | 与 spec 一致 |
| 受控组件 | 输入与展示双向绑定无「闪回旧值」 |

---

## 4. 文案与国际化

| 验收点 | Then 判据示例 |
| --- | --- |
| 关键文案 | 与产品文案表一致；无占位符 `undefined` |
| 多语言（若有） | 切换语言后全文覆盖关键路径；日期数字格式正确 |

---

## 5. 前端缓存：读 / 写 / 刷新

| 验收点 | Then 判据示例 |
| --- | --- |
| SWR / localStorage / IndexedDB | 过期与手动刷新后数据与后端一致 |
| 写后读 | 创建/更新成功后列表与详情**无**需整页刷新才一致（或 Then 写明必须刷新） |
| 隐私模式 / 清除站点数据 | 行为可接受或提示用户（按 spec） |

---

## 6. 向后端传递数据

| 验收点 | Then 判据示例 |
| --- | --- |
| 请求体字段名与类型 | 与 OpenAPI/契约一致；**无**静默丢字段 |
| 时区与日期 | UTC/本地与后端约定一致；边界日不含糊 |
| 文件上传 | 大小、类型、多文件顺序；失败重传 |

---
