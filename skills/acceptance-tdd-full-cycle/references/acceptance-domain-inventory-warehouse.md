# 领域验收：库存与仓储

**适用**：涉及「库存 / 仓储」时展开；与订单、支付链路上的扣减顺序须在 spec 或所链接的验收文档中 **只在一处写清**，他处仅交叉引用。

---

## 1. 库存扣减与回滚

| 验收点 | Then 判据示例 |
| --- | --- |
| 扣减与回滚 | 下单扣减库存；订单取消/退款库存回滚；无扣减错误 |
| 并发扣减 | 同一商品并发下单无超卖；库存与订单状态一致 |
| 预占与释放 | 预占库存按时效释放；超时订单库存回滚 |

---

## 2. 多仓路由与分配

| 验收点 | Then 判据示例 |
| --- | --- |
| 多仓分配 | 按配置规则分配库存；优先扣减最近仓库 |
| 仓库切换 | 仓库切换时库存数据同步；无数据不一致 |
| 跨仓调拨 | 调拨后库存数据正确；调拨日志可追溯 |

---

## 3. 盘点与预警

| 验收点 | Then 判据示例 |
| --- | --- |
| 盘点流程 | 盘点后库存数据正确；差异数据处理符合规范 |
| 库存预警 | 库存低于阈值时预警生效；预警通知正确 |
| 盘点差异处理 | 盘点差异数据可追溯；处理流程符合规范 |

---

## 4. 权限与审计

| 验收点 | Then 判据示例 |
| --- | --- |
| 仓库权限 | 不同角色/用户可操作的仓库不同；权限控制生效 |
| 库存操作日志 | 库存扣减、回滚、调拨等操作可追溯；关键字段不泄露敏感信息 |
